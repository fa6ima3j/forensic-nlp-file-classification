# Detecting File Fragmentation Using NLP & Data Classification

## Overview
This project explores how natural language processing (NLP) and data classification techniques can be used to analyse fragmented files in digital forensics, improving efficiency and reliability.

## Problem
Fragmented files make it difficult to reconstruct data quickly and accurately, impacting the reliability and integrity of forensic investigations.

## Solution
- Applied NLP techniques to analyse fragmented data  
- Used classification models to identify patterns  
- Evaluated model performance across different approaches  

## Tech Stack
- Python  
- Natural Language Processing (NLP)  
- Machine Learning  

## Testing & Validation
- Compared multiple models and algorithms  
- Evaluated performance using different metrics  
- Tested consistency and accuracy of results  
- Iteratively improved model performance  

## Key Outcomes
- Improved understanding of fragmented data analysis  
- Identified effective classification techniques  
- Highlighted the impact of access time on forensic efficiency
